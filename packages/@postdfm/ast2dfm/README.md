# @postdfm/ast2dfm

> Sub-package for [`postdfm`](https://www.npmjs.com/package/postdfm).
>
> Provides functionality for turning the structure provided by [`@postdfm/ast`](https://www.npmjs.com/package/@postdfm/ast) into a string/DFM.
